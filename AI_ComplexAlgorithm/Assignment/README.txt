To run this program you need to run it with arguments. It supports 2 arguments and both have to be present.
First argument is name of algorithm you want to use. Second is amount of colours you want to use.

List of algorithm names, used as first argument:

	randomSearch
	restartHillClimber
	iterLocalSearch

These should be self explanatory

Additional algorithms that are useful for debugging and testing, can be used as first argument 

	singleHillClimber
	randomSollution
